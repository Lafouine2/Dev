HTML

HTML est utilisé pour structurer une page web.

HTML signifie HyperText Markup Language. Il permet de créer des titres, des paragraphes, des liens, des images et bien plus encore.

Apprendre plus sur HTML

CSS

CSS sert à styliser et mettre en forme une page web.

CSS (Cascading Style Sheets) est utilisé pour ajouter du style aux pages HTML, comme les couleurs, les polices et la mise en page.

Apprendre plus sur CSS

JavaScript

JavaScript ajoute des interactions aux pages web.

JavaScript est un langage de programmation qui permet de rendre les sites web interactifs, comme les animations et les formulaires dynamiques.

Apprendre plus sur JavaScript

C

Le langage C est puissant et très utilisé en programmation système.

C est un langage de programmation bas niveau souvent utilisé pour le développement de systèmes d'exploitation et d'applications embarquées.

Apprendre plus sur C

Java

Java est un langage populaire pour les applications mobiles et les logiciels.

Java est un langage orienté objet utilisé pour le développement d'applications d'entreprise, mobiles (Android) et de logiciels.

Apprendre plus sur Java

Python

Python est un langage simple et puissant pour le développement web et l'IA.

Python est un langage très populaire pour le développement web, l'intelligence artificielle, la data science et l'automatisation.

Apprendre plus sur Python

C++

C++ est une extension du C avec des fonctionnalités orientées objet.

C++ est un langage très utilisé pour les jeux vidéo, les moteurs graphiques et les logiciels nécessitant de hautes performances.

Apprendre plus sur C++

C#

C# est un langage utilisé pour le développement d'applications et de jeux.

C# est principalement utilisé pour développer des applications Windows et des jeux avec Unity.

Apprendre plus sur C#

PHP

PHP est un langage serveur utilisé pour créer des sites dynamiques.

PHP est un langage de programmation côté serveur utilisé pour créer des sites dynamiques comme WordPress et Facebook.

Apprendre plus sur PHP

SQL est un langage servant à exploiter des bases de données relationnelles.

SQL (Structured Query Language) est un langage normalisé pour mettre à jour, récupérer et calculer des données dans les tables d'une base de données.
